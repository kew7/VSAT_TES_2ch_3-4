========================================================================
Project of Visual Studio 2019 Community.

Project for receiving and processing signals of satellite modems of the VSAT-TES Quantum system by Hughes NS.

Исходные коды проекта по приему и обработке сигналов спутниковых модемов системы VSAT-TES фирмы Hughes NS.
Сигналы вводятся в ПЭВМ через интерфейсное устройство ввода на базе Xilinx и TMS (далее драйвер),
на вход которого поступают демодулированные сигналы  с модемов, далее - обработка и регистрация
сигнала в соответствии с указанными ниже параметрами сигнала:
QPSK-РВК16-НСК3/4-V.35
(демодуляция QPSK - перемежитель РВК16 (цикл 16 бит) - сверточный декодер НСК 3/4 - дескремблер ITU-T V.35 (3,20)).
Общая информация по сетям VSAT: https://www.osp.ru/nets/2000/04/141088 
                                https://satprocom.ru/dokumentacija/13402/ .
